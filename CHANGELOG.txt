V39.4 — 27.04.2026
====================

ИСПРАВЛЕНО
-----------

1. Главная: при обновлении страницы внутри товара больше не возвращает
   на тот же товар.
   Раньше: пользователь зашёл по ссылке ?item=123, потом закрыл карточку,
   листал каталог, нажал F5 — снова открывалась та же карточка.
   Теперь: при закрытии карточки ?item=... автоматически убирается из URL,
   F5 даёт чистую страницу. Параметр ?cat=... сохраняется — категория
   остаётся выбранной.

2. ВК-сообщение: если артикула нет — строка "Арт:" вообще не пишется
   (раньше было "Арт: —").

3. Повторный клик по кнопке "🔧 Фильтры" — закрывает фильтр.
   Открыл, посмотрел, тыкнул в ту же кнопку — закрылось. Удобно.

ИЗМЕНЕНО
---------

4. Ссылка "Посмотреть на Авито →" в карточке заменена на
   "🔗 Скопировать ссылку на товар".
   Стиль такой же. По клику копирует ссылку вида
   https://artemcernysev12-svg.github.io/mebel-favorit/?item=4155668774
   и показывает "✅ Ссылка скопирована" на 2 секунды. Теперь её можно
   быстро отправить клиенту в любом мессенджере.
