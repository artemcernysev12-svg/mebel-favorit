   - index.html              (главное — он обновился сильнее всего)

   - robots.txt              (новый путь к sitemap)
   - sw.js                   (без изменений)
   - og-preview.jpg          (без изменений)
   - CHANGELOG.txt           (новый — что поменялось)
   - INSTRUKTSIYA_STATISTIKA.txt  (новый — как подключить аналитику)
   - README.txt              (этот файл)

   Папки можно НЕ ЗАГРУЖАТЬ — они не менялись:
   - catalog.js              (не менялся)
   - catalog-data/           (не менялась)
   - stock-data/             (не менялась)

   GitHub предложит заменить существующие файлы — соглашайся.

4. Внизу страницы напиши коммит, например:
   "Update to V39: VK fix, sitemap, admin panel, URL fix"
5. Жми "Commit changes".

Способ Б — через git (если умеешь):

   cd mebel-favorit-v39
   cp -r * /путь/к/локальному/репозиторию/
   cd /путь/к/локальному/репозиторию
   git add .
   git commit -m "V39: VK fix, sitemap, admin panel, URL fix"

ОБНОВЛЕНИЕ КАТАЛОГА В БУДУЩЕМ
==============================

Когда нужно будет обновить цены, добавить товары или прислать новую таблицу:

1. Замени файл catalog.js на новый.
2. Открой index.html, найди строку: catalog.js?v=V39
3. Поменяй на v=V40 (или любую другую новую цифру) — это нужно,
   чтобы у клиентов в браузерах подтянулась новая версия каталога.
4. Закоммить и запушь.

Если что-то не получается — пиши, помогу.
