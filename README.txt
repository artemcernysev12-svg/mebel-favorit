МЕБЕЛЬ ФАВОРИТ — сайт для GitHub Pages
========================================

Сайт работает по адресу: https://мебель-фаворит.рф

ЧТО ГДЕ ЛЕЖИТ
==============

В КОРНЕ — служебные файлы, которые НЕЛЬЗЯ удалять или перемещать:

  index.html           — главная страница сайта
  CNAME                — настройка домена для GitHub Pages
  catalog.js           — каталог товаров (1915 позиций)
  catalog-data/        — модули кухонь и прихожих (грузятся по запросу)
  stock-data/          — файл наличия (stock.xlsx)
  assets/              — логотипы (logo-icon, logo-header, logo-main)
  og-preview.jpg       — превью при шаринге в ВК/Telegram
  robots.txt           — настройки для поисковиков
  sitemap.xml          — карта сайта для поисковиков
  sw.js                — service worker (PWA)
  yandex_*.html        — подтверждение Яндекс.Вебмастера
  googlec*.html        — подтверждение Google Search Console

ПАПКА docs/ — справочные документы (только для тебя, не для сайта):

  START_HERE.txt              — с чего начать (читай первым)
  README.txt                  — общий обзор сайта
  INSTRUKTSIYA_STATISTIKA.txt — как пользоваться Метрикой и админкой
  CHANGELOG.txt               — что нового в каждой версии


КАК ОБНОВИТЬ ОСТАТКИ
=====================

Просто положи свежий файл выгрузки 1С (или из Б24) в:
  stock-data/stock.xlsx
и закоммить в репо. Сайт сам подхватит.


КАК СМЕНИТЬ ADMIN_CODE
=======================

Открой index.html в редакторе, найди:
  const ADMIN_CODE = 'mebel2026';
Замени 'mebel2026' на свой пароль. Закоммить.

Админ-панель открывается по адресу:
  https://мебель-фаворит.рф/?admin=твой_пароль
