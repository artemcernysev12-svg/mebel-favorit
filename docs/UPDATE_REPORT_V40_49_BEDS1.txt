V40_49_BEDS1 — первый этап: только кровати

Что изменено:
1. Правки сделаны от чистой версии v40_47, без изменений по кухням и шкафам.
2. В основной модалке кровати общий статус может показывать количество:
   - только кровать: В наличии — N шт.
   - кровать + основание/настил/опции: В наличии — N компл.
   - если не хватает деталей: Под заказ
3. В окне выбора опций кровати у вариантов основания теперь видно В наличии / Под заказ.
4. У доп. опций Подъёмный механизм и Короб/дно внутри окна выбора также видно В наличии / Под заказ.
5. В блок Итог комплектации не добавлялись технические строки по деталям — он оставлен в прежней логике.

Не трогали:
- кухни / Агава
- шкафы-купе
- общие клиентские статусы сайта
- ручную кнопку загрузки остатков
