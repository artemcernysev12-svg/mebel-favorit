V40_48 — правки наличия и подписей

Исправлено:
1. Кровати с выбором опций: при варианте «только кровать» и других вариантах наличие теперь выводится в шт., а не в компл.
2. Шкафы-купе Эра/Гранд/Элегант: наличие теперь считается по корпусу и фасадам и участвует в общем фильтре «В наличии»/бейдже на карточке.
3. Для шкафов-купе убрана подпись «Под заказ / уточнить наличие деталей» — теперь при отсутствии наличия выводится просто «Под заказ».
4. Для комплектов/составов убрана подпись «Нет данных по части модулей» при загруженных остатках — если артикул не найден в остатках, показывается «Под заказ».
5. Для кухни Агава и кухонных модулей добавлена проверка наличия по артикулу из состава, даже если модуль не связан с отдельной карточкой каталога.
6. В блоке «Собрать из модулей» статус модуля теперь рассчитывается по артикулу из строки состава: «В наличии» или «Под заказ».
7. Обновлены cache-busting версии подключений до V40_48.
