V40_49 — исправление наличия кухонь Агава

Что изменено:
1. Для модулей Агава наличие теперь считается не только по корпусу, а по формуле: корпус + фасад.
2. Если корпус есть, но фасада нужного цвета/типа нет — модуль показывает «Под заказ».
3. Для карточек кухонь Агава общий статус наличия также считается через обязательные компоненты с учётом фасадов.
4. Цвет Агавы в конструкторе теперь берётся из «Цвет от производителя», чтобы корректно различать Трюфель, Лиственница светлая, Антрацит и другие цвета.
5. Кеш-версия обновлена до V40_49.
